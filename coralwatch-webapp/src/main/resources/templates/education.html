Education Package

After 18 months of collaborative development between the University of Queensland, Brisbane, Australia and local marine educators, the Reef Education Package is now available for order. The Package provides a unique opportunity for your group to

  1. Learn about bleaching in the classroom or field
  2. Use the Chart in the field to monitor coral bleaching

The complete package costs AUD$160 (inc. GST) and contains:

  • Class set (30) of Coral Health Charts
    • The charts can be used to monitor coral reefs by simply matching the colour of corals to the chart. For more information see www.coralwatch.org
  • A Guide to Using the Coral Health Charts (hard copy)
    • 21-page document outlining how you can use the charts in a ‘classroom’ setting
  • CD containing
    • A Guide to Using the Coral health Charts (Microsoft Word format for editing and printing as required)
    • Reef Fingerprint© and Reef Transect© Microsoft Excel spreadsheets for data analysis
    • Virtual Reef (Microsoft Powerpoint presentation)
    • Virtual Lab (Microsoft Powerpoint presentation)
  • Virtual Transect poster (140 x 60cm)
  • Hard copy of the images in the Virtual Reef and Virtual Lab
  • Sample dive/snorkel data slate

The basic package costs AUD$110 (inc. GST) and contains:
